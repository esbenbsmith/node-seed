# Seed Template for Node.js with Express, and Body-parser

## What's in the Box?

1. Node.js
2. Express
3. Underscore (server)
4. Body-parser
5. Underscore.js (client)
6. Jquery
7. Boostrap



## Getting Started

1. ```$ git clone git@github.com:henryfreel/node-seed.git```
2. ```$ cd node-seed```
3. ```$ npm install```
4. ```$ open index.html```
