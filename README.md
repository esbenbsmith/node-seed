# Seed Template for Node.js with Express, and Body-parser

## What's in the Box?

1. Node.js
2. Express
3. Underscore (server)
4. Body-parser
5. Underscore.js (client)
6. Jquery
7. Boostrap



## Getting Started

1. ```$ git clone https://github.com/ajbraus/seed-jquery.git seedJquery```
2. ```$ cd seedJquery```
3. ```$ open index.html```
